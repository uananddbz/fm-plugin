Fm-plugin

jquery pugin for file manager

Download this project as a .zip file Download this project as a tar.gz file

installation

  1. we need below files - fm.js , ajax_fm.php

  2. include js - <script src="fm.js"></script>

  3. call plugin -
    <script type="text/javascript"> $(document).ready(function(){ $("element").fm("folder_name"); }); </script>
